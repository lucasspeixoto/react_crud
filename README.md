## Prerequisite
* Nodejs v14+
* Reactjs
* Axios
* Bootstrap
* Font Awesome
* React Router
* React Router DOM

## Getting Started
```sh
git clone https://github.com/lucasspeixoto/crud.git

cd crud

npm install

npm start
```

FrontEnd - Open [http://localhost:3000](http://localhost:3000)<br>
BackEnd - Open [http://localhost:3001](http://localhost:3001)<br>
